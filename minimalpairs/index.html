Minimal Pairs

What features does a model use to differentiate concepts?

Select two images to compare. The first bar plot shows how minimal changes to Image A changes the probability of the model predicting Name(A). The second bar chart shows how minimal changes to A changes the probability of the model predicting Name(B).

Each bar corresponds to the model's behavior on a different counterfactual image. The bar is labeled by the changes made to A to generate the counterfactual image; e.g., if the color was changed from red to blue, "red->blue". Hovering over a bar will show you the corresponding counterfactual image below.
